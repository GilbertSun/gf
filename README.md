gf (A JavaScript FrameWork Written By GilbertSun)
====

## 介绍

结合公司的前端需求和自己的兴趣爱好，尝试构建一套前端的组件库，希望能够借此提高自己的开发水平并给公司做贡献

## 包含的组件

* tab 传统的tab页切换组件
* carousel 轮播组件，可扩展为viewpage展示，并支持移动端
* dropdown 下拉菜单组件
* palette 调色版的组件

## 项目结构

```
root/
  |-- dist     项目编译的目标
  |-- doc      项目的文档
  |-- examples 项目中的演示例子
  |-- src      项目中中司机的模块代码
  `-- package.json
```
